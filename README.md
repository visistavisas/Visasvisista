# Visista Visas Landing Page

A responsive landing page for **Visista Visas** designed to promote work visa opportunities for New Zealand and Ireland.

## Project Overview

This landing page includes visa program information, destination highlights, a customized contact form, appointment booking fields, resume upload, and an embedded Thank You page after form submission.

## Features

- Responsive landing page design
- Visista Visas JPG logo in header and footer
- New Zealand and Ireland work visa content
- Contact form with:
  - Full name
  - Phone number
  - Email address
  - Preferred destination
  - Preferred appointment date
  - Preferred appointment time
  - Resume upload
  - Message field
- Resume upload validation
- Embedded Thank You page
- Visit Website button
- Email Us popup with contact email
- Mobile-friendly layout
